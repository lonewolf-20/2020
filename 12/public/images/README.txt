- mask.png

Before running the game, please ensure these image assets are properly placed in this directory. 
Placeholder or custom images can be used as long as they maintain the same filenames.

For development purposes, you can use any 32x32 pixel (items) or 32x48 pixel (characters) images 
with transparent backgrounds.
